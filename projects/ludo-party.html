OVERVIEW

Ludo Party is a modern multiplayer adaptation of Vietnam’s traditional Cờ Cá Ngựa board game, reimagined with vibrant 3D visuals and online multiplayer gameplay. While preserving the familiar dice-based movement and strategic race mechanics, the game introduces a new Funny Mode featuring unpredictable board effects, dynamic player interactions, and cosmetic customization.


Players compete against friends or online opponents to guide all of their pieces safely to the finish line while strategically eliminating rivals, avoiding hazards, and adapting to random board events. The game combines classic board game strategy with modern social features, progression systems, and live multiplayer gameplay.

MY ROLE & RESPONSIBILITIES

This was a solo project — I designed and built the entire game, including its multiplayer backend, covering:

  • Game Design: Designed the multiplayer turn-based board game logic and rules, adapted faithfully from the classic Ludo game.
  • Networking: Implemented real-time multiplayer using Photon Fusion, handling turn synchronization and reconnection.
  • Backend Integration: Integrated Firebase for player accounts, matchmaking data, and persistence.
  • Social Features: Built features such as friend invites and in-game chat/emotes to enhance player engagement.

CASE STUDY

The Challenge

Reliably synchronize turn-based board game state across multiple players in real time, while handling disconnects gracefully.

The Approach

Used Photon Fusion's networked state authority model to keep game state consistent, with reconnection logic to handle players dropping mid-match.

The Result

Smooth multiplayer sessions with minimal desync issues, allowing groups of friends to play a full match together reliably online.
